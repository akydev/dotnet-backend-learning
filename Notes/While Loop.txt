1. while Loop

The while loop checks the condition first, and only runs the code if the condition is true.

Syntax:

while (condition)
{
    // code to execute repeatedly
}


Example:

int i = 1;

while (i <= 5)
{
    Console.WriteLine("Number: " + i);
    i++;  // increase i by 1 each time
}

Output:
Number: 1
Number: 2
Number: 3
Number: 4
Number: 5
